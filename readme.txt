Adds a dashboard widget that displays the total word count for each public post type, plus a sortable column in the post list.

== Description ==

**Simple Word Counter Dashboard** is a lightweight plugin that helps authors and site administrators track the amount of content published across different post types.

Features:
- Dashboard widget with word count per post type.
- "Words" column added to post list screen.
- Word count saved in post meta for better performance.
- Sortable post list column.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the WordPress admin panel.
3. Visit your Dashboard to see the word count widget.
4. Go to "Posts" to see the new "Words" column.

== Frequently Asked Questions ==

= Can I use this for custom post types? =  
Yes! The plugin detects all public post types.

= Is the column sortable? =  
Yes, you can sort your posts by word count.

= Does it support multilingual sites? =  
Currently, it works in English only, but it can be extended with translation support.
