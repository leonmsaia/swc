A lightweight plugin that shows the total word count for each post type in the WordPress dashboard. Also adds a sortable "Words" column in the post list.

== Description ==

Simple Word Counter Dashboard is a WordPress plugin that helps you keep track of your content production.

* Adds a dashboard widget with the total word count for each post type.
* Displays a "Words" column in the post list.
* Sortable column for quick comparison.
* Saves word count in post meta for better performance.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin from the "Plugins" screen.
3. View the widget in the WordPress Dashboard.
4. Open the "Posts" admin screen to see the new "Words" column.

== Frequently Asked Questions ==

= Does it work with custom post types? =
Yes, it detects and includes all public post types.

= Is the word count calculated each time? =
No. It's saved in post meta and updated on post save for performance.
